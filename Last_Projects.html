MIT App Inventor Project: Get the Gold

Partner: Alex Kingori

Start: 9/23/19 Deadline: 10/9/19

The process for making this app wasn’t that hard. To determine what changes to make in each version, we just ranted on everything wrong on the app. We also tried to make the app more interesting which is also how we determined the versions. For example, we increased the speed so that way the PirateShip moves faster. This makes the app more responsive and fun to play. To add on, some functions wouldn’t work like the screens. The screens would work for the ai companion app. If you do the screens for the apk file, the app doesn’t go that screen. If the apk file was able to go to the different screens, then we had a starting screen and an instruction screen. Overall, the process of making our app wasn’t that hard. All we had to do was make the app more interesting and fun to play to dictate the versions.
